# Astromine

A single-file incremental/idle game about galactic mining, trade and refining.
Pure vanilla JavaScript, HTML5 Canvas — **no build step, no dependencies, no
framework**. The entire game lives in `index.html`.

Ein Single-File-Idle-/Inkrementalspiel über galaktischen Bergbau, Handel und
Veredelung. Reines Vanilla-JavaScript, kein Build-Schritt, keine Abhängigkeiten.

## Play / Spielen

- **Web:** open `index.html` in any modern browser (double-click).
- **Online:** _GitHub Pages link here_ · _itch.io link here_

The game runs entirely in the browser; progress is saved in `localStorage`.
Languages: English (default) / Deutsch — switchable in the in-game settings.
